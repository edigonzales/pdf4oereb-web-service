Help

Images: If the images are embedded, it will use these images. If there are no embedded images, it will use the WMS GetMap requests.

Resolution: The rubberband, the north arrow and the scalebar look blurry if the provided images (embedded and wms) are not 300dpi.

Language: The static parts of the text do always correspond to the chosen language. The dynamic parts will only show up if the chosen language is available in the xml. If the language cannot be found in the xml, it will take the first language element found.

API: curl -XPOST -F language=it -F file=@CH344982777421_geometry_wms.xml https://pdf4oereb-vccf8.ondigitalocean.app/convert --output CH344982777421_geometry_wms.pdf
